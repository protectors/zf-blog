## 问题描述
	 
	 根据它的要求输出相应的字符。


## 思路
判断这个字符是否为数字，如果是不断叠加s的值，如果不是数字，则输出题目中相对应的字符，并重置s的值。

## 代码

```c++
#include <cstdio>
#include <cstring>
#include <iostream>
#include <cctype>
using namespace std;
int main()
{
    char a[200];
    int s=0,i,len,j;
    while(gets(a))
    {
        len=strlen(a);
        for(i=0;i<len;i++)
        {
            if(a[i]>='0'&&a[i]<='9')
                s+=a[i]-'0';
            else
            {
                if(a[i]=='!')
                    printf("\n");
                for(j=0;j<s;j++)
                    if(a[i]=='b')
                        printf(" ");
                    else
                        printf("%c",a[i]);
                s=0;
            }
        }
        cout<<endl;
    }
    return 0;
}
```

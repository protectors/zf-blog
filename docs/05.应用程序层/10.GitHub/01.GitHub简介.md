## GitHub简介
GitHub公司总部位于美国旧金山，拥有一只不知是章鱼还是猫的吉祥物octocat

- GitHub提供Git仓库的托管服务
- GItHub使用广泛

:::tip
GitHub与Git的区别：
- 在Git中，开发者将源代码存入名为`Git`的资料库中，并加以使用。
- GitHub则是在网络上提供Git仓库的一项服务。

:::

## HTTP简介
HTTP是万维网的应用层通信协议，Web页面中的所有数据都是通过这个简单的文本协议进行传输的。

HTTP是一种无状态，由文本构成的请求-响应协议，这种协议使用的是客户端-服务器计算模型
